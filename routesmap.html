Descripción

Una aplicación para obtener la ubicación en la que se encuentra el usuario, puede marcar dos puntos y se marcara una ruta en el mapa para llegar al otro punto, o marcar tres puntos en diferentes lugares, y ver la información donde se encuentra el punto que marco en el lugar seleccionado, ciudad y país.

Tecnologias

Este proyecto se realizó con el lenguaje de Kotlin, retrofit, haciendo uso de la API openrouteservice.org que se utilizó para obtener los datos del mapa junto con GoogleMaps.
